# GoPro Live Tilt Viewer

A web-based tool to visualize GoPro video telemetry. Upload a GoPro video, and it will extract the accelerometer data to show you the real-time tilt angle during playback.

## How to Launch

1. **Install Node.js**
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
